How to use
1. install python3 (https://www.python.org/downloads/)
2. Add path to installed Python folder to PATH (=system environment variable) (may require reboot)
3. Run Terminal in this folder
4. python ./runMain.py

사용법 
1. 파이썬 3 설치 (https://www.python.org/downloads/)
2. 설치된 파이썬 폴더의 경로를 PATH(=시스템 환경 변수)에 추가하기 (재부팅이 필요할 수 있음)
3. 이 폴더에서 터미널 열기 (빈 공간 우클릭, 터미널에서 열기)
2. python ./runMain.py 입력
